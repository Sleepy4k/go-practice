# Golang Practice

Learn basic syntax for go language

## Run Locally  

Clone the project

~~~bash  
  git clone https://github.com/Sleepy4k/go-practice.git
~~~

Go to the project directory

~~~bash  
  cd go-practice
~~~

Build the go file

~~~bash  
  go build <filename>
~~~

Start the go file

~~~bash  
  ./<filename>
~~~

## Alternatif

~~~bash
  go run <filename>
~~~
